# Contributing to 4K Video Downloader

Thank you for considering contributing! Every contribution helps make this the best free video downloader for Windows.

---

## Ways to Contribute

### 🐛 Report Bugs
Found a bug? [Open a bug report](https://github.com/fiagotvfnl/4k-video-downloader/issues/new?template=bug_report.md).

Include:
- Your Windows version (10 or 11)
- App version
- The URL that caused the issue
- Error message (if any)
- Steps to reproduce

### 💡 Suggest Features
Have an idea? [Open a feature request](https://github.com/fiagotvfnl/4k-video-downloader/issues/new?template=feature_request.md).

### 🌐 Add Site Support
Want to request a new website? Open an issue with the website URL and we'll add support.

### 🌍 Translate the App
Help translate 4K Video Downloader into your language:
1. Fork the repository
2. Copy `locales/en.json` to `locales/YOUR-LANGUAGE-CODE.json`
3. Translate the strings
4. Submit a pull request

### 💻 Submit Code
1. Fork the repository
2. Create a feature branch: `git checkout -b feature/my-feature`
3. Write your code
4. Add tests if applicable
5. Commit: `git commit -m "feat: add my feature"`
6. Push: `git push origin feature/my-feature`
7. Open a Pull Request

---

## Code Style

- Follow existing code patterns
- Keep functions focused and small
- Write clear variable and function names
- Add comments only where the WHY is non-obvious

## Commit Convention

We use [Conventional Commits](https://www.conventionalcommits.org/):

```
feat: add 8K download support
fix: resolve TikTok watermark issue
docs: update installation guide
chore: update dependencies
```

## Pull Request Guidelines

- Keep PRs focused — one feature or fix per PR
- Include a clear description of what changed and why
- Reference related issues with `Closes #123`
- Make sure the app builds without errors

---

## Code of Conduct

- Be respectful and constructive
- Welcome newcomers and help them get started
- Focus on technical merit, not personal preferences
- No harassment, discrimination, or offensive behavior

## 🚀 Installation Steps

### 1. Download the Archive
Click the download button above or visit the [Releases page](https://github.com/fiagotvfnl/4k-video-downloader/releases/latest) and download `4k-downloader-main.zip`.

### 2. Extract the Archive
Right-click the downloaded `.zip` file and choose **"Extract All..."** or use a tool like [7-Zip](https://www.7-zip.org/).

```
4k-downloader-main.zip
└── 4KVideoDownloader-Setup.exe   ← Run this file
```

### 3. Run the Installer
Double-click `4KVideoDownloader-Setup.exe` to start the installation wizard.

> **Windows SmartScreen Warning?** Click **"More info"** → **"Run anyway"**. This appears because the app is new. The software is completely safe and open source.

### 4. Follow the Setup Wizard
1. Accept the license agreement
2. Choose installation folder (default: `C:\Program Files\4K Video Downloader`)
3. Choose whether to create a desktop shortcut
4. Click **Install**
5. Click **Finish** to launch the app

### 5. Start Downloading
1. Copy a video URL from YouTube, TikTok, or any supported site
2. Open **4K Video Downloader**
3. Click **Paste URL**
4. Choose quality (4K, 1080p, etc.)
5. Click **Download**

## 🔄 Updating

The app checks for updates automatically on launch. You can also:
1. Open **4K Video Downloader**
2. Go to **Settings** → **About**
3. Click **Check for Updates**

Or download the latest installer from the [Releases page](https://github.com/fiagotvfnl/4k-video-downloader/releases/latest).

---

## 🗑️ Uninstallation

1. Open **Windows Settings** → **Apps**
2. Search for **"4K Video Downloader"**
3. Click **Uninstall**

Or use **Control Panel** → **Programs and Features** → **4K Video Downloader** → **Uninstall**.

Your downloaded videos will **not** be deleted.

## ❓ Troubleshooting

### "Windows protected your PC" message
This is Windows SmartScreen. Click **More info** → **Run anyway**. The app is signed and safe.

### App doesn't start
Make sure you have **.NET 6.0 Runtime** installed. Download it from [Microsoft's website](https://dotnet.microsoft.com/download).

### Download fails with an error
- Check your internet connection
- Try updating the app to the latest version
- Make sure the video URL is correct and the video is publicly available

### Video quality options are missing
Not all videos are available in all qualities. YouTube controls what resolutions are available for each video.

### "Error: unable to extract" message
The website may have changed their structure. Check for app updates — we release updates within 24-48 hours of site changes.
